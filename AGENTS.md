# Agent Instructions

- Canonical project instructions are maintained in `.agents/AGENTS.md`. Read and follow that file as the source of truth.
- Read `.agents/settings.json` at the start of each task for repo, branch, and stack-specific settings.
- When a matching skill exists in `.agents/skills/`, read that `SKILL.md` plus its `references/context.md` and `templates/checklist.md`.
